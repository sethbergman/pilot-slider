[pilot-slider.npmstack.com](http://pilot-slider.npmstack.com)

### Setup

```
git clone https://github.com/sethbergman/pilot-slider.git && cd pilot-slider
```

### Install dependencies

```
npm install
```

### Start the app

```
npm start
```

#### Visit <http://0.0.0.0:5000> in your browser.
